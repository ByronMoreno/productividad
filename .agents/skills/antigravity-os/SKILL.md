# Antigravity OS - Sistema de Productividad Personal

Este skill define la filosofía fundamental, arquitectura, y fases de desarrollo del sistema operativo personal diseñado para mitigar la fatiga mental y maximizar la ejecución sin fricciones.

## Filosofía Fundamental
- **"Tu cerebro está hecho para crear, no para recordar pendientes."**
- **"Nunca pensar qué hacer después."** (El sistema prioriza, decide y presenta la acción óptima según el contexto y nivel de energía actual; el usuario solo ejecuta).
- **Reducción de Ansiedad**: Ocultar la lista masiva de tareas pendientes y mostrar únicamente la tarea del momento o el bloque del día.

## Módulos del Sistema
1. **Inbox**: Captura ultrarrápida y sin clasificar de pensamientos, ideas y tareas.
2. **Clasificador IA**: Procesamiento automático del Inbox (detecta proyecto, área, tipo, tiempo estimado, prioridad, deadline).
3. **Kanban Inteligente**: Flujo dinámico autoorganizado (Pendiente -> Hoy -> En Proceso -> Esperando -> Hecho).
4. **Time Blocking Automático**: Construcción inteligente del calendario del día sin arrastrar bloques manualmente.
5. **Motor Antiestrés**: Oculta la complejidad; muestra solo lo realizable hoy.
6. **Sistema de Energía**: Clasificación de tareas por costo de energía (1 a 5 estrellas). Ajuste automático del día según el cansancio del usuario.
7. **Context Awareness**: Integración con el calendario real (clases, reuniones) para sugerir bloques adecuados de trabajo.
8. **Metas y Objetivos Anuales**: Conexión vertical desde metas anuales hasta las microtareas del día.
9. **Dashboard de Enfoque**: Indicadores de tiempo profundo, interrupciones, tiempo perdido y nivel de estrés.
10. **IA Coach**: Diálogo interactivo para desbloquear tareas o redefinir prioridades bajo alta fatiga.
11. **Modo Concentración**: Interfaz zen minimalista (tarea actual + checklist + temporizador).
12. **Módulo Antiprocrastinación**: Detección de inactividad (20 mins) para ofrecer alternativas de replanificación o descanso.
13. **Revisión Semanal IA**: Balance de tareas completadas/abandonadas y sugerencias de límites de trabajo.
14. **Base de Conocimiento ("Segundo Cerebro")**: Vinculación de soluciones técnicas, notas y repositorios con tareas resueltas.
15. **Botón "Brain Off"**: Automatización total del día. La IA genera el itinerario y el usuario solo pulsa "Empezar" y avanza en secuencia.

## Fases de Desarrollo
- **Fase 1 (Actual)**: Captura (Inbox), Proyectos, Kanban básico y Time Blocking.
- **Fase 2**: Motor de IA para priorización automática por urgencia, importancia y energía disponible.
- **Fase 3**: Gestión de Conocimiento vinculada (Notas, código y documentación asociados a tareas).
- **Fase 4**: Analíticas de productividad, monitor de interrupciones y sugerencias inteligentes de hábitos.
- **Fase 5**: Agentes de IA especializados por dominio (Docencia, Inversiones, Investigación, Programación).
