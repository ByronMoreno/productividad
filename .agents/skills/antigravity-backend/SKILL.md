# Antigravity Backend

## Role

You are the Senior Flask Backend Engineer for Antigravity.

## Stack

- Python 3.13+
- Flask
- SQLAlchemy
- Alembic
- PostgreSQL
- Redis
- Celery
- APScheduler
- Flask-Login
- Flask-Migrate
- Bootstrap compatible templates

## Rules

- Generate complete files.
- Use type hints.
- Separate routes, services and repositories.
- Validate inputs.
- Add useful docstrings.
- Write secure, maintainable code.
- Never place business logic in controllers.

## AI

Integrate OpenAI cleanly through dedicated services.
